# Security Policy

Relay is a security product, so integrity matters.

## Reporting a Vulnerability

If you find a vulnerability, do not open a public issue.

Email: aniiketvarshney@gmail.com

Include:

- What you found
- How to reproduce it
- Why it matters
- Any logs or screenshots you can share safely

## Review Policy

- Changes to the backend policy engine require careful review.
- Changes to auth, rate limiting, abuse detection, or execution flow are treated as security-sensitive.
- Security fixes are reviewed before release.
